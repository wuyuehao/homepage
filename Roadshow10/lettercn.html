Apache 2010 亚洲巡演将于2010年8月14日至15日在中国上海举行。

Apache巡演是一个由Apache软件基金会主办的全球性会议。Apache软件基金会成立于1999年，是一个的非盈利性组织。它所运行的Apache开源社区是全世界开源社区中规模最大并且最为成功的社区之一。它的大多数项目涉及企业计算的很多方面，尤其体现在Web应用和Java平台等领域。Apche软件基金会为独立开发者和企业开发了许多脍炙人口的软件项目，包括Apche HTTP服务器、Subversion版本控制系统、Harmony Java运行库、Tomcat Web容器、Geronimo Java EE应用服务器、Struts Web框架、Derby数据库等等。

每年，Apache软件基金会都会在世界各地举行会议探讨开源软件的发展方向，介绍Apache项目的最新进展，为开源软件开发者和用户提供一个相互交流的平台。Apache的会议被公认为各类开源社区举办的会议中最高水平的会议之一。Apache亚洲巡演是Apache在亚洲的会议名称。去年的Apache亚洲巡演在北京举行，Apache软件基金会主席Justin Erenkrantz、副主席及Subversion创始人之一Greg Stein和中国开源软件推进联盟主席陆首群等出席了会议。参与者有来自Intel、IBM、Google等知名企业和来自Apache、Mozilla、Eclipse等社区的开发者。更多详情请访问这里。

2010年的会议的主题是开源软件和新兴技术，面向软件从业人员、高校师生和所有的开源爱好者。目的是提供一个展示和交流的平台来介绍Apache和其他开源组织的项目的最新进展以及社区活动等方面的信息。它集开源软件开发人员、用户、公司等于一堂，共同探讨技术、共享经验、交流思想、结交朋友!

Keynotes演讲者:

   Justin Erenkrantz 目前担任Apache软件基金会主席。之前他是Apache的财务。他目前就职于Project WBS。他对多个软件项目作出过重大贡献，包括Apache HTTP服务器、APR、mod_mbox、flood和Serf。他拥有美国加州大学的博士学位，毕业论文题为可计算的REST：一种分散的Internet量级的应用。

   Tim Ellison 目前是IBM英国Java技术中心的高级技术组成员。他在IBM全球范围负责Java SDK的开源工程，为IBM旗舰产品的广泛选择提供了坚实的基础。在此之前，Tim还在IBM加拿大、OTI、Continuum公司等担任过技术领袖职务。他是Apache软件基金会的VP和Apache Harmony项目管理委员会的主席。Tim拥有计算机科学学士学位，并在曼彻斯特大学获得计算机系统设计硕士学位。Tim在过去的二十多年中对于Smalltalk的商业实现、IBM VisualAge的微型版本、Eclipse以及Java SDK都做出过贡献。他在高性能运行库、开源方法论领域知识渊博，乐于使用一些面向对象技术的新方法来解决难题。Tim的夫人是Natalie，他们有四个孩子。

   臧斌宇 是复旦大学软件学院院长。他感兴趣的研究方向包括编译器、计算机体系结构以及系统软件解决方案。解决方案涵盖优化和并行化编译器、运行时优化、软件系统在线更新、系统虚拟化以及利用虚拟技术的可信计算、数据密集型计算体系等。他有多篇文章发表在国际会议上，例如MICRO、ISCA、ICSE、ICPP、LCPC、CC、LCTES和VEE。他是第4届可信云基础设施技术国际会议项目委员会的联席主席。他也是ACM SIGPLAN/SIGBED 2007、2008嵌入式系统的语言、编译器及工具会议(LCTES'07、LCTES'08)项目委员会成员。
